# Number Guessing Game

A simple Java-based Number Guessing Game where the computer generates a random number and the user attempts to guess it.

## Features
- Random number generation
- User input handling
- Hint messages (Too High / Too Low)
- Tracks guessing attempts
- Interactive console-based gameplay

## Technologies Used
- Java
- Scanner Class
- Random Class
- Loops
- Conditional Statements

## How It Works
1. The system generates a random number.
2. The user enters a guess.
3. The program provides hints:
   - Too High
   - Too Low
4. The game continues until the correct number is guessed.
5. The total number of attempts is displayed.
